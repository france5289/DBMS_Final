# DBMS_Final
DBMS final project : Movie forum
## Entity List
- Movie : 紀錄電影資訊
  - **name**
  - Rating (分級)
    - 18+
    - 15+
    - 12+
    - 6+
    - 0+
  - Tomato_score
  - IMDB_score
  - Box
- Producer : 紀錄製片商資訊
  - **name**
  - addr
  - founded_year
- User : 使用者資訊
  - **id**
  - level
  - Sex
- Platform : 觀影平台資訊
  - **name**
  - official_web
  - Category
    - theater
    - streamer   
- Director
  - **name**
  - BDate
  - Sex
  - Age
