# DBMS_Final
DBMS final project : Movie forum
## Entity List
- Movie : 紀錄電影資訊
  - **Movie_name**
  - Rating (分級)
    - 18+
    - 15+
    - 12+
    - 6+
    - 0+
  - Tomato_score
  - IMDB_score
  - World_Box
- Producer : 紀錄製片商資訊
  - **PName**
  - Country
  - Founded
- User : 使用者資訊
  - **id**
  - BDate
  - Sex
- Platform : 觀影平台資訊
  - **Plat_name**
  - official_web
  - Category
    - theater
    - streamer   
- Director
  - **DName**
  - Sex
  - BDate
